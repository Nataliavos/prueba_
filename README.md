INVENTORY AND SALES MANAGEMENT SYSTEM WITH REPORTS

Comprehensive inventory and sales management system for a bookstore, with dynamic reports based on sales made.

SYSTEM OBJECTIVE: this robust system allows:
-Recording sales and products
-Applying discounts according to customer type
-Validating available stock and updating it automatically.
-Generating detailed sales reports

inventory_system/
├── main.py        # Entry point, main menu
├── models.py      # Models (products, sales, discounts)
├── utils.py       # Utility functions (validations, messages)
├── inventory.py   # Product CRUD (inventory)
├── sales.py       # Sales registration and query
└── reports.py     # Reports module
